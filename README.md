# SmartERP

A cloud-based **Billing, Inventory & Accounting Management System** inspired by **Tally ERP**, built using modern web technologies.

## 🚀 Project Overview

SmartERP is an Enterprise Resource Planning (ERP) application that helps businesses manage:

- 📦 Inventory Management
- 💰 Accounting
- 🧾 Billing & Invoicing
- 🛒 Sales Management
- 📥 Purchase Management
- 👥 Customer Management
- 🏢 Supplier Management
- 🏦 Banking
- 📊 GST Management
- 📈 Financial Reports

The application is designed to provide a **keyboard-first workflow**, similar to Tally, while taking advantage of modern web technologies.
